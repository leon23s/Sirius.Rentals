# Документация
## API Сириус.Аренда
- Адрес: `http://127.0.0.1:5000`
- Входные данные: JSON, время в формате ISO
---
## Запуск Docker контейнера
1. откройте командную строку
2. перейдите в корневую папку проекта `cd C:\Users\...`
3. создайте папку для базы данных `mkdir db`
4. выполните команду для сборки и запуска контейнера `docker-compose up --build -d`
5. дождитесь загрузки зависимостей (примерно 1-2 минуты)
6. перейдите по адресу указанному выше
7. для остановки контейнера выполните команду `docker-compose down`
---
## Аутентификация
- `POST /login` - получить токен.
- Токен передавать в заголовке: `Authorization: Bearer <токен>`.
- Токен живёт 1 час.
---
## Эндпоинты
### Пользователи
- `POST /register` - создать пользователя (`email`, `password`, `name`)
### Комнаты
- `GET /rooms` - список комнат. необязательные параметры: `?capacity=10&equipment=проектор,доска`
- `POST /rooms` - добавить комнату
- `GET /rooms/<id>` - данные комнаты + активные брони.
- `PUT /rooms/<id>` - изменить комнату 
- `DELETE /rooms/<id>` - удалить комнату (только если нет активных броней)
### Бронь
- `GET /rooms/available?start=...&end=...&capacity=...` - свободные комнаты на промежуток времени.
- `GET /rooms/<id>/bookings?date=...` - брони на конкретную дату.
- `POST /bookings` - создать бронь (`room_id`, `date_start`, `date_end`)
- `DELETE /bookings/<id>` - отменить бронь (только владелец)
